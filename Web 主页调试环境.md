职业技能等级认证考试系统 - Web 主页调试环境

1. 系统入口页面 ： system-entry.html
   
   - 整合了所有功能模块的入口
   - 提供系统状态监控
   - 包含调试工具快速访问
   - 支持 Alt+D 快捷键打开调试控制台
2. 快速登录页面 ： quick-login.html
   
   - 无需密码快速登录各角色账号
   - 支持考生、教师、管理员三种角色
   - 可通过 URL 参数直接指定角色
3. 角色演示页面 ：
   
   - 考生界面： student-demo.html
   - 教师界面： teacher-demo.html
   - 管理员界面： admin-demo.html
   - 考试界面： exam-demo.html
4. 调试工具 ：
   
   - API 测试工具： api-tester.html
   - 调试面板： debug-dashboard.html
5. 主页入口 ： index.html
   
   - 添加了调试入口
   - 支持 Alt+D 快捷键显示/隐藏调试入口
   - 保留了原有的 Vue 应用挂载点
6. 使用说明 ： README.md
   
   - 提供了系统概述
   - 详细的调试指南
   - 功能模块说明
   - 快捷键和调试技巧
## 功能模块
系统包含以下主要功能模块，每个模块都有对应的调试入口：

1. 用户管理模块 ：用户增删改查、角色权限管理、考生管理
2. 题库管理模块 ：题库分类管理、试题编辑与审核、试题导入导出
3. 试卷管理模块 ：试卷创建与编辑、试卷模板管理、试卷发布、考试计划安排
4. 考试监控模块 ：实时监控、考试控制、防作弊系统、考试状态查看
5. 考生考试模块 ：考试界面、答题功能、断网处理、交卷功能
6. 成绩管理模块 ：成绩统计与分析、成绩导出、证书管理
7. 阅卷中心模块 ：自动阅卷、手动阅卷、评分规则设置
8. 日志审计模块 ：系统日志、用户活动记录、安全审计
9. 数据分析模块 ：考试数据分析、试题分析、报表生成、数据可视化
10. API网关模块 ：API管理、API认证、API监控
## 使用方法
1. 启动本地服务器： python -m http.server 8000
2. 访问系统入口： http://localhost:8000/frontend/system-entry.html
3. 或者通过 http://localhost:8000/frontend/index.html?debug=true 显示调试入口
## 调试技巧
- 使用快速登录页面无需密码直接登录各角色账号
- 使用 API 测试工具可以测试后端 API 接口
- 使用 Alt+D 快捷键可以快速显示/隐藏调试入口
- 系统入口页面提供了所有功能模块的直观访问
