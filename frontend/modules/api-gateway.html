API文档
POST 用户登录
/api/v1/auth/login
用户登录接口，验证用户身份并返回访问令牌。支持用户名密码登录和短信验证码登录两种方式。
🔧 所属服务: 用户服务
🔒 认证要求: 无需认证
⏱️ 超时设置: 5秒
请求参数
参数名 类型 是否必须 描述
username string 必须 用户名或手机号
password string 必须 密码，使用密码登录时必须
loginType string 可选 登录类型，可选值：password(密码登录)、sms(短信验证码登录)，默认为password
smsCode string 可选 短信验证码，使用短信验证码登录时必须
rememberMe boolean 可选 是否记住登录状态，默认为false
请求示例
{ "username": "admin", "password": "password123", "loginType": "password", "rememberMe": true }
响应参数
参数名 类型 描述
code integer 状态码，200表示成功
message string 状态消息
data object 响应数据
data.token string 访问令牌
data.refreshToken string 刷新令牌
data.expiresIn integer 令牌有效期（秒）
data.userInfo object 用户信息
响应示例
{ "code": 200, "message": "登录成功", "data": { "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...", "refreshToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...", "expiresIn": 3600, "userInfo": { "userId": "1001", "username": "admin", "realName": "管理员", "avatar": null, "roles": ["admin"], "permissions": ["*:*:*"] } } }
响应状态码
状态码 描述
200 登录成功
400 请求参数错误
401 用户名或密码错误
429 请求过于频繁，请稍后再试
500 服务器内部错误
