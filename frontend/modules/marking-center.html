问题 3：简述Java中的多态性及其实现方式

请简要描述Java中的多态性概念，并详细说明Java中实现多态的三种方式，每种方式请给出简单的代码示例。

（本题15分）

评分标准：
- 正确描述多态概念（3分）
- 列举三种实现方式（每种2分，共6分）
- 每种方式提供正确代码示例（每种2分，共6分）
考生答案：

Java中的多态性是指同一个行为具有多个不同表现形式或形态的能力。多态是同一个接口，使用不同的实例而执行不同操作。

Java多态性实现方式：

1. 继承实现多态：通过继承父类并重写父类方法

class Animal {
    void sound() {
        System.out.println("动物发出声音");
    }
}

class Dog extends Animal {
    @Override
    void sound() {
        System.out.println("狗叫声");
    }
}

Animal animal = new Dog();
animal.sound(); // 输出：狗叫声

2. 接口实现多态：通过实现接口并实现接口方法

interface Shape {
    double getArea();
}

class Circle implements Shape {
    private double radius;
    
    public Circle(double radius) {
        this.radius = radius;
    }
    
    @Override
    public double getArea() {
        return Math.PI * radius * radius;
    }
}

Shape shape = new Circle(5);
double area = shape.getArea();

3. 方法重载实现多态：同一个类中方法名相同但参数不同

class Calculator {
    int add(int a, int b) {
        return a + b;
    }
    
    double add(double a, double b) {
        return a + b;
    }
    
    int add(int a, int b, int c) {
        return a + b + c;
    }
}
