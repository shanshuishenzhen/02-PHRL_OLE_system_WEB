# 职业技能等级认证考试系统 - 前端调试指南

## 系统概述

职业技能等级认证考试系统是一个完整的在线考试平台，支持题库管理、试卷管理、考试监控、成绩管理、阅卷中心等功能模块。本文档提供了前端调试的相关指南。

## 调试入口

系统提供了多个调试入口页面，方便开发人员进行手动调试：

- **系统入口**：`system-entry.html` - 包含所有功能模块的入口
- **快速登录**：`quick-login.html` - 无需密码快速登录各角色账号
- **调试面板**：`debug-dashboard.html` - API接口调试工具
- **API测试工具**：`api-tester.html` - 测试后端API接口

## 角色演示页面

系统提供了三种角色的演示页面：

- **考生界面**：`student-demo.html` - 考生相关功能演示
- **教师界面**：`teacher-demo.html` - 教师相关功能演示
- **管理员界面**：`admin-demo.html` - 管理员相关功能演示

## 功能模块

系统包含以下主要功能模块：

1. **用户管理模块**
   - 用户增删改查
   - 角色权限管理
   - 考生管理

2. **题库管理模块**
   - 题库分类管理
   - 试题编辑与审核
   - 试题导入导出

3. **试卷管理模块**
   - 试卷创建与编辑
   - 试卷模板管理
   - 试卷发布
   - 考试计划安排

4. **考试监控模块**
   - 实时监控
   - 考试控制
   - 防作弊系统
   - 考试状态查看

5. **考生考试模块**
   - 考试界面
   - 答题功能
   - 断网处理
   - 交卷功能

6. **成绩管理模块**
   - 成绩统计与分析
   - 成绩导出
   - 证书管理

7. **阅卷中心模块**
   - 自动阅卷
   - 手动阅卷
   - 评分规则设置

8. **日志审计模块**
   - 系统日志
   - 用户活动记录
   - 安全审计

9. **数据分析模块**
   - 考试数据分析
   - 试题分析
   - 报表生成
   - 数据可视化

10. **API网关模块**
    - API管理
    - API认证
    - API监控

## 快捷键

系统提供了以下快捷键：

- `Alt+D`：显示/隐藏调试入口

## 调试技巧

1. **快速登录**：使用 `quick-login.html` 页面可以无需密码直接登录各角色账号

2. **API测试**：使用 `api-tester.html` 可以测试后端API接口

3. **URL参数**：
   - `index.html?debug=true`：显示调试入口
   - `quick-login.html?role=student`：直接进入考生快速登录
   - `quick-login.html?role=teacher`：直接进入教师快速登录
   - `quick-login.html?role=admin`：直接进入管理员快速登录

4. **调试控制台**：在 `system-entry.html` 页面中，点击"调试控制台"按钮或按 `Alt+D` 可以打开调试控制台

## 后端API

后端API基础路径：`http://localhost:8000/api/`

主要API模块：

- 用户管理：`/api/user/`
- 题库管理：`/api/question/`
- 试卷管理：`/api/paper/`
- 考试监控：`/api/monitor/`
- 成绩管理：`/api/score/`
- 阅卷中心：`/api/marking/`
- 日志审计：`/api/log/`
- 数据分析：`/api/analysis/`

详细API文档可访问：`/api/docs/`

## 本地开发

1. 确保后端服务已启动
2. 访问 `index.html?debug=true` 显示调试入口
3. 选择需要调试的功能模块进行测试

## 注意事项

1. 调试页面仅用于开发和测试环境，请勿在生产环境中使用
2. 所有操作将被记录在调试日志中
3. 快速登录功能仅在开发环境中有效
